⚡ COMECE EM 5 MINUTOS (Rodar Localmente)
───────────────────────────────────────────────────────────────────────────────
1. Abra PowerShell nesta pasta
2. Python -m venv venv
3. venv\Scripts\activate
4. pip install -r requirements.txt
5. python app.py
6. Abra http://localhost:5000
